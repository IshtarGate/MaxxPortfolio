Medstar Bugfixes

As part of our daily standup, a member of the team is assigned to manually test one of the domains of the MedStar HealthÂ wordpress multisite.

The result of this, is an internal audit of the Medstar Health multisite, where small visual, performance, and functional bugs are logged and queued for remedy.

I have begun to take on a number of small bug fixes.


For instance:

A small visual bug was logged in which header text on 320px viewport, small mobile phones, overflowed off the side of the page. An image of the bug is below.




To fix this bug, I simply tracked down the class name of the header, paying attention to the nesting of the class with other classes. This enabled me to add an entry to "responsive.scss", a file that contains CSS changes for mobile devices, which enacted a media query at 320px reducing the size of the font.
